# The Oracle

A deterministic predictor, written in Rust, that classifies an AI-generated
code change as **Pass** or **Fail** against a ground-truth test oracle.

## Build

```bash
cargo build --release
```

Requires a recent stable Rust toolchain (see `rust-toolchain.toml`).
