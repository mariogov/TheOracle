# O*NET Attribution

DynamicJEPA career-taxonomy runs use the public O*NET 30.2 text database as a
taxonomy source for generated benchmark fixtures. The generated fixture is a
research-demo artifact, not real recruiting history.

The local bundle `data_manifest.json` records the exact O*NET source ZIP path,
SHA-256, row count, selected occupation count, and generated fixture hashes.

License and source links:

- O*NET Database license: https://www.onetcenter.org/license_db.html
- O*NET Database downloads: https://www.onetcenter.org/database.html
- O*NET 30.2 text ZIP: https://www.onetcenter.org/dl_files/database/db_30_2_text.zip

Attribution boundary recorded in generated bundles:

O*NET 30.2 Database, Creative Commons Attribution 4.0 International License
(CC BY 4.0). Generated fixture modifies O*NET database information for a public
taxonomy benchmark; USDOL/ETA has not approved, endorsed, or tested these
modifications.
